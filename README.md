# DevelopersHub Phase-II Internship Tasks

# AI/ML Project Portfolio

## Overview

This repository contains three machine learning and deep learning projects demonstrating the application of Natural Language Processing (NLP), Machine Learning Pipelines, and Multimodal Deep Learning. Each project focuses on solving a real-world problem using modern AI techniques and industry-standard tools.

---

## Task 1: News Topic Classifier Using BERT

This project fine-tunes the **BERT (bert-base-uncased)** transformer model to classify news headlines into predefined topic categories using the **AG News dataset**. The workflow includes data preprocessing, tokenization, model fine-tuning, performance evaluation using **Accuracy** and **F1-Score**, and deployment through **Streamlit** for interactive predictions.

**Technologies:** Python, PyTorch, Hugging Face Transformers, Scikit-learn, Streamlit

---

## Task 2: End-to-End Machine Learning Pipeline for Customer Churn Prediction

This project builds a reusable and production-ready machine learning pipeline to predict customer churn using the **Telco Customer Churn dataset**. Data preprocessing, feature encoding, scaling, model training, and hyperparameter tuning are implemented using Scikit-learn's **Pipeline API** and **GridSearchCV**. The final pipeline is exported using **Joblib** for future deployment.

**Technologies:** Python, Pandas, Scikit-learn, Joblib

---

## Task 3: Multimodal Machine Learning for Housing Price Prediction

This project predicts housing prices by combining **house images** with **structured tabular data**. Image features are extracted using a **Convolutional Neural Network (CNN)** and fused with tabular features to train a multimodal regression model. The model is evaluated using **Mean Absolute Error (MAE)** and **Root Mean Squared Error (RMSE)**.

**Technologies:** Python, TensorFlow/Keras, OpenCV, Scikit-learn
